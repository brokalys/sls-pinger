## Requirements

- Node
- Yarn

## Installation

```sh
yarn install
```

## Testing

```sh
yarn test
```

## Linting

```sh
yarn lint
```

## Deployment

Github actions take care of deployments. The following command can be used for doing manual deployments.

```sh
yarn deploy
```
